CORBA IDL Test Suite V0.1
#########################

Consists of a comprehensive set of CORBA IDLs that can be used to test the
CORBA specification-compliance of any CORBA compiler.
These IDLs test compliance with version 2.3 of the OMG CORBA specification.

Revision History
================
0.1    Initial revision. Consists of 180 IDL files that test various portions
       of a CORBA compiler. IDL files are designed for CORBA 2.3 compliance.
